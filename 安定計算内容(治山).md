## はじめに(本書の位置付け)

### 本資料の目的

本ドキュメントは、鋼製スリットダムWEB設計システム開発(以下、本開発)における、治山基準の安定計算の計算内容を定義したドキュメントである。

### 注意点

本資料は 安定計算内容(砂防) の内容が前提となっており、異なる部分のみを記載している。

## 土石流時・鋼製部(クーロン)

### 土圧係数

$$ Ka = \frac{\cos^2(φ-α)}{\cos^2α \cdot \cos(α + δ) \cdot \{ 1 + \frac{\sin(φ + δ) \cdot \sin(φ + β)}{\cos(α + δ) \cdot \cos(α - β)} \}} $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">φ</div>：堆砂土の内部摩擦角<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">α</div>：壁背面の勾配角(度)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">δ</div>：壁背面と背面土の摩擦角(度)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">β</div>：渓床勾配(度)<br>
<br>

### 自重及び載荷重(自重は変更なし)

#### 堆積土部

- 土石流の水深が基礎部にかかる場合(H1' < h1)

$$ Wd1 = 1/2\times Li\times h2\times\gamma d $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Wd1</div>：基礎上流側傾斜部の土石流下端より下の堆積土の自重(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Li</div>：土石流下端での基礎前面の傾斜幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">h2</div>：土石流下端の高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd</div>：堆積土の空中単位体積重量(kN/m<sup>3</sup>)<br>
<br>

- 土石流の水深が基礎部にかからない場合(h1 < H1')

$$ Wd1 = 1/2\times(Lb - Li)\times(h3 + Ld)\times\gamma d $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Wd1</div>：スリット部上流側前面傾斜部の上にある堆積土の自重(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Lb</div>：スリット部前面の傾斜部の幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Li</div>：土石流下端での基礎前面の傾斜幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">h3</div>：基礎部上面の上流側端部から土石流下端までの高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Ld</div>：基礎の上流側張り出し幅における上面傾斜高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd</div>：堆積土の空中単位体積重量(kN/m<sup>3</sup>)<br>

$$ Wd2 = Lc\times h3\times\gamma d $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Wd2</div>：基礎前面からダム軸までの間の基礎部上面の上流側端部より上にある四角形部分の堆積土の自重(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Lc</div>：基礎部前面からダム軸までの幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">h3</div>：基礎部上面の上流側端部から土石流下端までの高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd</div>：堆積土の空中単位体積重量(kN/m<sup>3</sup>)<br>

$$ Wd3 = 1/2\times L2\times Ld\times\gamma d $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Wd3</div>：基礎部上面の上流側端部からダム軸までの間の基礎上にある三角形部分の堆積土の自重(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">L2</div>：基礎の張出し(上流側)(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Ld</div>：基礎の上流側張り出し幅における上面傾斜高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd</div>：堆積土の空中単位体積重量(kN/m<sup>3</sup>)<br>

$$ Wd4 = 1/2\times Lh\times H2'\times\gamma d' $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Wd4</div>：基礎部上流側傾斜部上にある三角形部分の堆積土の自重(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Lh</div>：基礎部前面の傾斜部の幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H2'</div>：基礎部上面の上流側端部での高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd'</div>：堆積土の水中単位体積重量(kN/m<sup>3</sup>)<br>

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γw</div>：水の単位体積重量(kN/m<sup>3</sup>)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">ΔH</div>：越流水深(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H0</div>：ダムの高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H2'</div>：鋼製部の基礎部上面の上流側端部での高さ(m)<br>

#### 堆積土による土圧力

$$ Pa1 = \gamma w\times (\Delta H + H0 - H2' )\times H2'\times Ka $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Pa1</div>：土石流より下の部分にかかる土石流による土圧力(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γw</div>：水の単位体積重量(kN/m<sup>3</sup>)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">ΔH</div>：越流水深(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H0</div>：ダムの高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H2'</div>：鋼製部の基礎部上面の上流側端部での高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Ka</div>：堆積土の土圧係数<br>
<br>

$$ Pa2 = 1/2\times\gamma d\times H2'^2\times Ka $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Pa2</div>：堆積土による土圧力(kN/m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">γd'</div>：堆積土の水中単位体積重量(kN/m<sup>3</sup>)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">H0</div>：ダムの高さ(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">Ka</div>：堆積土の土圧係数<br>

### アーム長の計算

#### 自重及び載荷重

$$ LWw1 = B - 1/2\times L3 $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">LWw1</div>：Ww1のアーム長(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">B</div>：袖断面下幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">L3</div>：袖部上流面下部の傾斜幅(m)<br>

$$ LWw2 = B - 1/3\times L3 $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">LWw2</div>：Ww2のアーム長(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">B</div>：袖断面下幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">L3</div>：袖部上流面下部の傾斜幅(m)<br>

$$ LWw3 = B - L3 - 1/3\times La $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">LWw3</div>：Ww3のアーム長(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">B</div>：袖断面下幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">L3</div>：袖部上流面下部の傾斜幅(m)<br>
<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">La</div>：袖部上流面に作用する越流水の三角形部分の幅(m)<br>

$$ LWd1 = B - 1/3\times Lh $$

<div style="margin-left: 0.5cm; width: 2cm; display: inline-block;">LWd1</div>：Wd1のアーム長(m)<br>
